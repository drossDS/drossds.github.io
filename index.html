Boston Airbnb Fair Pricing Tool and Recommender


Developed a pricing model to estimate Boston Airbnb listing prices. Used web scraping, APIs, and natural language processing (NLP) techniques to engineer features. Implemented a recommender system using cosine similarity to suggest listings based on user inputs. The pricing model outperformed the null model by reducing the root mean squared error from $86 to $49 and achieved an R-Squared value of 0.68 on validation data with outliers removed.

...Or click here to expand a brief summary!
Key Accomplishments
  • Data were collected from Inside Airbnb for over 3800 Airbnb listings in the Boston Area in 2023 with 75 features provided for each listing.
  • A feature to simulate the distance of each listing to Boston subway stations was created by web scraping station addresses from the Massachusetts Bay Transit Authority (MBTA) website before using the Google Geocoding API to return latitude and longitude coordinates for each listing.
  • Additional features were engineered which used Natural Language Processing (NLP) Techniques to examine descriptive text columns.
  • The correlation coefficients between all features and listing price were used to select the best features on which to train the model.
  • Seven different regression models were run, and the best models were optimized using grid searching. Principal Component Analysis (PCA) was also attempted to reduce dimensionality and improve price predictions.
  • A simple recommender employing cosine similarity was created to find listings most-similar to simulated user-input listing criteria.
Key Outcomes
  • Models had a large impact on the model performance with the best models initially achieving a root mean squared error (RMSE) of $389 and an R-Squared value of 0.09 on validation data.
  • Performance improved drastically by removing listings with prices below the 5th percentile and above the 95th percentile to an RMSE of $49 and an R-Squared value of 0.68. This is an improvement over the null model’s RMSE of $86 with outliers removed.
  • The best performing model was a random forest regressor.
  • A recommender was capable of correctly recommending listings which met the simulated user-input listing criteria.

Detemining the Severity of Alzheimer's Disease from MRI Images

Built image classification models employing both convolutional and pre-trained neural network models to classify the severity of Alzheimer’s disease in MRI images. The model achieved an accuracy of 71% across four severity classes and improved upon the null model accuracy of 50%. A second binary classification model was developed focusing on detecting very early stages of dementia by training on only two severity classes, achieving an accuracy of 81%, an improvement over the null model accuracy of 59%.
